# Assumptions and Requirements

## Assumptions

- The campus consists of three institutes.
- All institutes share a centralized data center.
- All services are hosted on Windows Server 2022.
- Users access resources through wired and wireless networks.
- The network supports future expansion.

## Network Requirements

- Support approximately 2500 devices.
- Provide Internet access.
- Provide centralized authentication.
- Provide centralized DNS services.
- Provide centralized DHCP services.
- Provide file sharing services.
- Provide a web portal using IIS.

## Security Requirements

- VLAN segmentation.
- Access Control Lists.
- Secure device management using SSH.
- Strong password policies.
- Network monitoring and auditing.

## Performance Requirements

- Scalable design.
- High availability.
- Efficient routing.
- Centralized management.

## Future Expansion

- Additional institutes can be added.
- Additional servers can be deployed.
- Wireless infrastructure can be expanded.
